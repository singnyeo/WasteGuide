# 🗑️ 올바른 쓰레기 배출 안내 애플리케이션
**👥 디투금쪽이들**

> 일상 속에서 발생하는 쓰레기를 올바르게 배출하고, 관련 법과 벌금 정보를 함께 안내하는 환경 보호 실천 앱


## 📱 소개

쓰레기 배출 기준을 잘 몰라 발생하는 **환경오염**, **주민 불편**, **벌금 문제**를 해결하고자 이 애플리케이션을 개발

- 쓰레기 종류별 분리배출 방법 안내  
- 불법 투기 시 벌금 및 관련 법률 정보 제공  
- 환경 관련 행사 및 기념일 소개

## 📌 주요 기능

- 생활 쓰레기 종류별 배출 방법 제공
- 잘못된 배출 시 발생 가능한 법적 처벌 기준 안내
- 환경 기념일, 캠페인 등의 정보 제공
- 사용자 친화적인 UI
